# FTC_Lesson_Plan
This is my software lesson plan for new and returning FTC Robotics students


Starting from day 1, the below are what we're going to train our students to do

1) Lesson 1: Install software and overview
   - Install and overview of Git
       - clone our repo
       - using the app
       - making changes/ pull request
   - Install and overview of Android Studio
       - accessing the code
       - compiling

2) Lesson 2: Basic Configuration
   - Android studio configuration
   - file configuration of robot defaults
   - Driver hub configuration of defaults
 
3) Lesson 3: Robot features and push software to the robot
   - Show basic mechanum robot and talk about parts
   - Use wire to load code to robot
       - Tools and toroubleshooting
   - Wirelessly load code to robot
   - Push code to the robot (control hub) (use default driving code)
   - Validating that the code pushed
   - Run the code
  
4) Lesson 4: Basic robot driving
   - Manualy drive the robot around (forward/backward, strafe left/right, rotate left/right)
   - Review the control of the wheels
   - Do the motor/wheel validation thing on driver hub
   - How to control speeds ( and acceleration curves)
  
5) Lesson 5: Basic Odometry
   - Learn about odometry pods (we use gobuilda)
   - Get position
   - Get heading
   - Zero heading and position
  
6) Lesson 6: My first AutoMode
   - Move forward, turn CW 90, straif right, turn 180, straif left, turn ccw 90, backup, turn 180.... did I end in my start point
   - How validate distance
   - how validate rotation angle
   - How control / validate directional speed
   - how control / validate rotational speed
  
7) Lesson 7: Obstical Course 1
